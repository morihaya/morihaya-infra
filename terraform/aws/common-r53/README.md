# common-r53

共通の Route 53 DNS レコードを管理するための Terraform ワークスペースです。

## 概要

このディレクトリは、`morihaya.tech` ドメインの共通 DNS レコードを Terraform Cloud で管理します。

## 管理対象

- Route 53 の DNS レコード（A、CNAME など）

## ワークスペース

- **HCP Terraform Workspace**: [morihaya-infra-aws-common-r53](https://app.terraform.io/app/morihaya/workspaces/morihaya-infra-aws-common-r53)
- **Organization**: `morihaya`
